Le code HTML est le code utilisé pour structurer une page web et son contenu. Chaque élément a sa balise attribuée : par exemple pour du texte il faut la balise :

  • Le code HTML est le code utilisé pour structurer une page web et son contenu. Chaque elements à sa balise atribué : par exemple pour du texte il faut la balise : <p> Ceci est un paragraphe </p> ou pour faire un titre il faut utiliser la balise:
    <h2> Ceci est un titre h2 </h2> (il y a 6 niveaux de titres).
  • Le code CSS est le code utilisé pour mettre en forme une page web. On crée des classes ou des ID qui ont des propriétés ou valeurs CSS pour styliser des éléments HTML. Par exemple, si on veut que notre texte soit bleu, on crée une classe :
    .bleu{color:#0000FF} et on l'applique à l'élément HTML désiré, par exemple:
    <p class="bleu"> Ceci est un paragraphe </p>
