Les types de variables

En JavaScript, il existe plusieurs types de variables :

  • Les variables numériques : elles permettent de stocker des nombres, entiers ou décimaux. Exemple : var x = 5;
  • Les variables de chaînes de caractères : elles permettent de stocker du texte. Exemple : var nom = 'Dupont';
  • Les variables booléennes : elles permettent de stocker des valeurs "vrai" ou "faux". Exemple : var estVrai = true;
  • Les variables null : elles permettent de définir une variable comme étant vide. Exemple : var y = null;
  • Les variables undefined : elles permettent de définir une variable comme n'ayant pas de valeur. Exemple : var z;
